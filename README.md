# DreamWeave AI — سایت تعبیر رویا با هوش مصنوعی

سایت کامل، زیبا، دو زبانه (فارسی + انگلیسی)، با بک‌اند FastAPI تک‌فایل.

## ویژگی‌ها
- طراحی premium مثل سایت‌های خارجی (تم تاریک mystical)
- ورودی متن + صوتی (Voice Input با مرورگر)
- تحلیل رویا قدرتمند مبتنی بر روانشناسی واقعی (بدون نیاز به API پولی)
- پشتیبانی از زبان‌های مختلف + پرچم کشورها
- سیستم کاربران واقعی + اشتراک ۵ سطحی (Free تا Very Excellent)
- تاریخچه رویاها + محدودیت روزانه
- مونتایز دستی با شماره کارت (قابل ویرایش)

## فایل‌ها
- `dreamweave_ai.py` → فایل اصلی (همه چیز داخلش: بک‌اند + فرانت‌اند کامل)
- `requirements.txt` → کتابخانه‌های مورد نیاز

## نحوه اجرا محلی (تست)
```bash
pip install -r requirements.txt
uvicorn app:app --reload
```
سپس در مرورگر برو به: http://127.0.0.1:8000

## نحوه Deploy روی Render (۱۰۰٪ کار می‌کنه)

### مرحله ۱: آماده‌سازی GitHub
1. یک ریپو جدید در GitHub بساز.
2. فایل‌های زیر رو آپلود کن:
   - `app.py` (فایل اصلی)
   - `requirements.txt`
   - `Dockerfile` (برای Deploy با Docker)

### مرحله ۲: تنظیمات مهم قبل از آپلود
**حتماً این کار رو بکن:**
- فایل `app.py` رو باز کن.
- بخش `PAYMENT_INFO` رو پیدا کن.
- `card_number` رو با **شماره کارت واقعی خودت** جایگزین کن.
- `card_holder` و توضیحات رو هم ویرایش کن.

### مرحله ۳: Deploy روی Render
1. برو به https://dashboard.render.com
2. New → Web Service
3. GitHub repo خودت رو وصل کن
4. تنظیمات:
   - **Runtime**: Python 3
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `uvicorn app:app --host 0.0.0.0 --port $PORT` (اگر از Docker استفاده می‌کنی، نیازی به این نیست)
   - **Plan**: Free (کافیه)
5. (اختیاری ولی توصیه می‌شه) یک Environment Variable بساز:
   - Name: `SECRET_KEY`
   - Value: یک رشته طولانی رندوم (مثلاً ۵۰ کاراکتر)

6. Deploy کن. بعد از چند دقیقه سایتت آنلاین می‌شه.

### مرحله ۴: فعال کردن پلن رایگان برای خودت (سازنده)
بعد از اولین Deploy:
1. با ایمیل خودت ثبت‌نام کن.
2. از Render Shell یا ابزار sqlite این دستور SQL رو اجرا کن:
   ```sql
   UPDATE users SET subscription = 'very_excellent' WHERE email = 'ایمیل_تو@gmail.com';
   ```
حالا تو همه چیز رو رایگان و نامحدود داری.

## نکات مهم

## Deploy با Docker روی Render (توصیه شده برای زبان و پایداری بیشتر)

Render از Docker به خوبی پشتیبانی می‌کنه و برای پروژه‌هایی مثل این خیلی بهتره (کنترل کامل محیط، زبان فارسی و ...).

### مراحل Deploy با Docker:

1. **فایل‌های مورد نیاز** (همه آماده هستن):
   - `dreamweave_ai.py` (تغییر نام به `app.py` در ریپو)
   - `requirements.txt`
   - `Dockerfile` (همین الان اضافه شده)

2. **در GitHub**:
   - ریپو رو آپدیت کن و این سه فایل رو آپلود کن.
   - فایل `dreamweave_ai.py` رو به `app.py` تغییر نام بده (یا در Dockerfile تنظیم کن).

3. **در Render**:
   - New → Web Service
   - GitHub repo رو انتخاب کن
   - این بار **Environment** رو روی **Docker** بگذار (نه Python)
   - Render خودش Dockerfile رو تشخیص می‌ده و می‌سازه.
   - Port: 8000 (یا Render خودش هندل می‌کنه)
   - Deploy کن.

4. **نکته مهم برای زبان فارسی**:
   - در Dockerfile از `python:3.11-slim` استفاده شده که کاملاً از UTF-8 و زبان فارسی پشتیبانی می‌کنه.
   - هیچ مشکلی برای متن فارسی، RTL و ... وجود نداره.

5. **Environment Variables** (اختیاری):
   - `SECRET_KEY` = رشته رندوم طولانی

بعد از Deploy، سایتت با Docker اجرا می‌شه و خیلی پایدارتره.
